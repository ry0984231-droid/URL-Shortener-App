# URL Shortener App
A full-stack URL Shortener application that allows users to shorten long URLs, track analytics, and manage links securely.
## Features
- User Signup/Login Authentication
- JWT & Cookie-based Authentication
- Password Hashing using bcrypt
- Short URL Generation
- URL Analytics Tracking
- MongoDB Database Integration
- ## Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- JWT
- bcrypt
- ## Installation
git clone https://github.com/ry0984231-droid/URL-Shortener-App.git
cd URL-Shortener-App
npm install
npm start
## Environment Variables
PORT=8002
MONGO_URL=your_mongodb_url
JWT_SECRET=your_secret
